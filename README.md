# ExamEdge

AI-powered exam prep tool built for the ColorStack x GDG Hackathon. Upload a past exam or your lecture notes and get back a focused study pack - top concepts, practice questions, and a cram sheet — generated by Gemini AI in seconds.


## What it does

**Mode 1 — Past Exam Analysis**
Upload a previous exam paper (PDF, image, or text). ExamEdge identifies the most tested concepts, generates new questions in the same style, and writes a focused cram sheet.

**Mode 2 — Notes to Practice Exam**
Upload lecture notes or a syllabus. ExamEdge generates 10 exam-style questions on the most important topics plus a complete cram sheet.

## Setup

### Prerequisites

- Python 3.10+
- A Google account (to get a free Gemini API key)

### 1. Get a Gemini API key

1. Go to [aistudio.google.com](https://aistudio.google.com)
2. Sign in with your Google account
3. Click **Get API key** → **Create API key**
4. Copy the key

### 2. Clone the repo

```bash
git clone https://github.com/ZainaNadeem/Team_Maverick.git
cd Team_Maverick
```

### 3. Set up the backend

```bash
cd Backend
```

Create a `.env` file (use `.env.example` as a template):

```bash
cp .env.example .env
```

Open `.env` and paste your key:

```
GEMINI_API_KEY=your_key_here
```

Install dependencies:

```bash
pip install -r requirements.txt
pip install google-genai
```

Start the server:

```bash
python -m uvicorn main:app --reload --port 8000
```

The API is now running at `http://localhost:8000`. You can verify it at `http://localhost:8000/health`.

### 4. Open the frontend

Open `frontend/index.html` directly in your browser, or serve it with Python for the best experience:

```bash
cd frontend
python -m http.server 3000
```

Then visit `http://localhost:3000`.


## How to use

1. Open the app in your browser
2. Choose a mode — **Analyse a past exam** or **Generate from notes**
3. Upload your file (PDF, PNG, JPG, or TXT — up to 20 MB)
4. Optionally add your subject/module name for more focused results
5. Click the button and wait a few seconds
6. Browse your study pack across three tabs: **Top Concepts**, **Practice Questions**, and **Cram Sheet**
7. Hit **Download PDF** to save it


## API endpoints

| Method | Endpoint | Description |
|---|---|---|
| GET | `/health` | Health check |
| POST | `/analyse` | Run Gemini analysis on uploaded content |

### POST /analyse

Request body:
```json
{
  "prompt": "full prompt string",
  "file_base64": "base64 encoded file (optional)",
  "mime_type": "application/pdf (optional)"
}
```

Response:
```json
{
  "result": "TOP CONCEPTS:\n...\nPRACTICE QUESTIONS:\n...\nCRAM SHEET:\n..."
}
```
